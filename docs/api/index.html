JsonDir

Manages JSON files in a directory.

new JsonDir(dir: String, options: JSON?)
Parameters
dir (String) Path of the JSON dir
options (JSON?) Options
Name Description
options.encoding String? Default is 'utf8'. Encoding of JSON files.
options.replacer String? Default is null. Either a function or an array used to transform the result. The replacer is called for each item.
options.space (String | Number)? Default is 2. A string to be used as white space (max 10 characters), or a Number, from 0 to 10, to indicate how many space characters to use as white space.
options.git String? Default for Windows is 'C:\Program Files\Git\bin\git.exe' and for Linux is '/usr/bin/git'. Path to git binary.
Example
const JsonDir = require(`@brokfeld/jsondir`);

const users = new JsonDir(`./users`);

▸ commit(file, options)

Executes git add and git commit on a JSON file.

commit(file: String, options: JSON): Promise<JSON>
Parameters
file (String) File name
options (JSON) JSON like { name: 'Max Mustermann', email: 'max@mustermann.de' }
Name Description
options.name String User name
options.email String User email
options.message String? Commit message
Returns
Promise<JSON>: JSON like { stdout: '', stderr: ''}
Example
const JsonDir = require(`@brokfeld/jsondir`);

(async () => {
  const users = new JsonDir(`${__dirname}/dir_with_git`);
  await users.write(`user1`, { random: Math.random() });
  await users.commit(`user1`, { name: `Max Mustermann`, email: `max@Mmustermann.de` });
})();

▸ find(func)

The find() method returns the value of the first JSON file in the JSON dir that pass a test (provided as a function).

find(func: Function): (Promise<JSON> | Promise<null>)
Parameters
func (Function) Function like (content, file) => { return true; }
Returns
(Promise<JSON> | Promise<null>):
Example
const JsonDir = require(`@brokfeld/jsondir`);

(async () => {
  const users = new JsonDir(`${__dirname}/users`);
  const result = await users.find((content, file)=>{
    console.log(file);
    return content.age === 25;
  });
  console.log(result); // { file: 'user1', content: { name: 'Tom', age: 25 } }
})();
